App Name - Medicare
Java e-healthcare web application for ordering medicines of different categories.
Developer Name: Shantha Kumar
Email: shanth24kumar@gmail.com

Available Scripts
npm install
Run this script in both, the root folder and the ExpressBackend folder to install all the dependencies.

npm start
After extracting the project, you first need to start the express backend server.
This will start the backend server at http://localhost:8080

Now, to start the frontend react server, navigate to the root folder, open the terminal there and then run 'ng serve' command.
This will start the frontend server at http://localhost:4200
